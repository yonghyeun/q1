# One-Pager (MVP)

작게 시작하기 위한 1장 요약. 길게 쓰지 않는다.

Terminology SoT: [glossary.md](glossary.md)

## 제품명(가제)

- unote

## 타깃(한 문장)

- 유튜브로 무엇인가를 공부하는 기록형 학습자가, 타임스탬프 기반으로 근거가 남는 노트를 빠르게 남기고 반복 사용하게 만든다.

## 문제(왜 지금 불편한가)

- 산만함: 추천/연관/쇼츠로 이탈한다.
- 분리: 시청과 기록이 분리돼 맥이 끊긴다.
- 근거 부재: 기록이 영상의 어느 지점을 근거로 하는지 남기기 어렵다.

## 해결(핵심 경험 1줄)

- 큐레이션한 채널 영상 피드(필터/더 보기 포함)에서 재생하거나 영상 URL로 바로 열고, 같은 화면에서 마크다운 노트를 쓰며 단축키로 타임스탬프를 남기고 클릭으로 해당 시점으로 점프한다.

## 차별점(딱 1~2개)

- 타임스탬프 기반 레퍼런싱이 "기본"이다(삽입/클릭 점프가 핵심 UX).
- 등록한 채널만 보여주는 집중 학습 환경으로 학습 세션을 보호한다.

## MVP 범위(In Scope)

- 채널 URL 등록
- 등록 채널 통합 피드(All) + 채널 필터 + `Load older`(30개씩)
- (탈출구) 유튜브 영상 URL 붙여넣기로 바로 열기
- Player + Note 화면(마크다운 노트: 작성/열람/수정, 자동저장)
- 타임스탬프 삽입/클릭 점프
- 회원가입/로그인 없이 사용(브라우저 로컬 저장)

## Out of Scope (이번 사이클에 안 함)

- 영상 다운로드/편집/클리핑
- 회원가입/로그인, 클라우드 동기화/백업
- 팀 협업(권한/코멘트)
- 추천 알고리즘/개인화 피드
- 플레이리스트 URL/ID 소스 등록(후속 MVP에서 구현 예정)

## 성공 지표(초기)

- Activation: 첫 세션에서 타임스탬프 삽입 1회 이상 비율
- Retention: D1 재방문, D7 재방문
