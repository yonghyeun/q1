## WBS Link (Optional)
- WBS-01

## Problem / Opportunity
- PR/브랜치 운영 규칙이 바뀌면서 기존 자동화 문서와 실제 스크립트가 불일치한다.

## Goal
- 단일 에이전트 + WBS 운영 기준에 맞춰 issue 생성 경로를 정리한다.

## In Scope
- 이슈 템플릿 정비
- issue 생성 스크립트 인자 정리

## Out of Scope
- PR 머지 자동화 정책 변경

## Acceptance Criteria
- [ ] 템플릿 본문 검증을 통과한다.
- [ ] issue 생성 스크립트가 정상 동작한다.

## Risks
- 기존 문서/스크립트와의 용어 차이로 혼선이 생길 수 있다.
