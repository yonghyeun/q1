# Decisions

의사결정 로그를 저장한다.

목표: 나중에 "왜 이렇게 했지?"를 설명 가능하게 만든다. 길게 쓰지 않는다.

## 파일 네이밍(권장)

- `YYYY-MM-DD_slug.md`
- 예: `2026-03-03_mvp-scope-cut.md`

## 템플릿

```md
# DEC: <title>

- Date: YYYY-MM-DD
- Context: (어떤 상황/문제였나)
- Decision: (무엇을 결정했나)
- Alternatives: (고려한 대안 1~2개)
- Tradeoffs: (무엇을 포기했나)
- Revisit if: (어떤 조건이면 재검토할까)
```
